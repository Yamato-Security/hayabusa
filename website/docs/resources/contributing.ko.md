# 기여 및 지원

## 기여

어떤 형태의 기여든 환영합니다.
풀 리퀘스트, 규칙 작성, 샘플 evtx 로그가 가장 좋지만 기능 요청, 버그 알림 등도 매우 환영합니다.

최소한 저희 도구가 마음에 드신다면 GitHub에서 별을 눌러 지지를 보여주세요!

## 버그 제출

발견하신 버그는 [여기](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)에 제출해 주세요.
이 프로젝트는 현재 활발히 유지보수되고 있으며 보고된 버그를 기꺼이 수정하겠습니다.

Hayabusa 규칙에서 문제(오탐, 버그 등)를 발견하시면 hayabusa-rules GitHub 이슈 페이지 [여기](https://github.com/Yamato-Security/hayabusa-rules/issues/new)에 보고해 주세요.

Sigma 규칙에서 문제(오탐, 버그 등)를 발견하시면 상위 SigmaHQ GitHub 이슈 페이지 [여기](https://github.com/SigmaHQ/sigma/issues)에 보고해 주세요.

## 라이선스

Hayabusa는 [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html)로 배포되며 모든 규칙은 [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md)로 배포됩니다.
Hayabusa를 내부적으로, SaaS 솔루션, 컨설팅 업무 등에 자유롭게 사용하실 수 있습니다.
다만 Hayabusa를 일종의 SaaS 솔루션에 사용하고 개선하는 경우, 그 개선 사항을 오픈소스로 공개하여 프로젝트에 기여해 주시기를 부탁드립니다.

Hayabusa는 MaxMind가 생성한 GeoLite2 데이터를 사용하며, 이는 [https://www.maxmind.com](https://www.maxmind.com)에서 이용할 수 있습니다.
