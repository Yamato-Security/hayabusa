# ပံ့ပိုးကူညီခြင်းနှင့် ပံ့ပိုးမှု

## ပံ့ပိုးကူညီခြင်း

ကျွန်ုပ်တို့သည် မည်သည့်ပုံစံဖြင့်မဆို ပံ့ပိုးကူညီမှုကို ကြိုဆိုပါသည်။
Pull request များ၊ rule ဖန်တီးခြင်းနှင့် နမူနာ evtx log များသည် အကောင်းဆုံးဖြစ်သော်လည်း feature request များ၊ bug များကို ကျွန်ုပ်တို့အား အသိပေးခြင်းစသည်တို့ကိုလည်း အလွန်ကြိုဆိုပါသည်။

အနည်းဆုံးအားဖြင့်၊ ကျွန်ုပ်တို့၏ tool ကို သင်နှစ်သက်ပါက ကျေးဇူးပြု၍ GitHub တွင် ကျွန်ုပ်တို့အား star တစ်လုံးပေးပြီး သင့်ပံ့ပိုးမှုကို ပြသပါ။

## Bug တင်ပြခြင်း

သင်တွေ့ရှိသော မည်သည့် bug ကိုမဆို ကျေးဇူးပြု၍ [ဤနေရာတွင်](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D) တင်ပြပါ။
ဤ project ကို လက်ရှိတွင် တက်ကြွစွာ ထိန်းသိမ်းနေပြီး အစီရင်ခံတင်ပြထားသော မည်သည့် bug ကိုမဆို ကျွန်ုပ်တို့ ဝမ်းမြောက်စွာ ပြုပြင်ပေးပါသည်။

Hayabusa rule များနှင့်ပတ်သက်၍ မည်သည့်ပြဿနာ (false positive များ၊ bug များ စသည်) ကိုမဆို သင်တွေ့ရှိပါက ၎င်းတို့ကို hayabusa-rules GitHub issues စာမျက်နှာ [ဤနေရာတွင်](https://github.com/Yamato-Security/hayabusa-rules/issues/new) ကျေးဇူးပြု၍ အစီရင်ခံတင်ပြပါ။

Sigma rule များနှင့်ပတ်သက်၍ မည်သည့်ပြဿနာ (false positive များ၊ bug များ စသည်) ကိုမဆို သင်တွေ့ရှိပါက ၎င်းတို့ကို upstream SigmaHQ GitHub issues စာမျက်နှာ [ဤနေရာတွင်](https://github.com/SigmaHQ/sigma/issues) ကျေးဇူးပြု၍ အစီရင်ခံတင်ပြပါ။
