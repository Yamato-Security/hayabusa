# Kontribusi & Dukungan

## Kontribusi

Kami sangat menyambut segala bentuk kontribusi.
Pull request, pembuatan rule, dan contoh log evtx adalah yang terbaik, tetapi permintaan fitur, memberi tahu kami tentang bug, dll... juga sangat kami hargai.

Paling tidak, jika Anda menyukai alat kami, mohon berikan kami bintang di GitHub dan tunjukkan dukungan Anda!

## Pengiriman Bug

Mohon kirimkan bug apa pun yang Anda temukan [di sini.](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)
Proyek ini saat ini dipelihara secara aktif dan kami dengan senang hati memperbaiki bug apa pun yang dilaporkan.

Jika Anda menemukan masalah (false positive, bug, dll...) dengan rule Hayabusa, mohon laporkan ke halaman GitHub issues hayabusa-rules [di sini](https://github.com/Yamato-Security/hayabusa-rules/issues/new).

Jika Anda menemukan masalah (false positive, bug, dll...) dengan rule Sigma, mohon laporkan ke halaman GitHub issues hulu SigmaHQ [di sini](https://github.com/SigmaHQ/sigma/issues).

## Lisensi

Hayabusa dirilis di bawah [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) dan semua rule dirilis di bawah [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md).
Anda dapat dengan bebas menggunakan Hayabusa secara internal, untuk solusi SaaS, pekerjaan konsultasi, dll...
Namun, jika Anda menggunakan Hayabusa dalam sebuah solusi tipe SaaS dan melakukan perbaikan terhadapnya, kami meminta Anda untuk membuka kode sumber (open-source) perbaikan tersebut dan memberikannya kembali ke proyek.

Hayabusa menggunakan data GeoLite2 yang dibuat oleh MaxMind, tersedia dari [https://www.maxmind.com](https://www.maxmind.com).
