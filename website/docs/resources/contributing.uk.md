# Внесок і підтримка

## Внесок

Ми будемо раді будь-якій формі внеску.
Pull-запити, створення правил і зразки журналів evtx — найкращі, але запити на нові функції, повідомлення про помилки тощо також дуже вітаються.

Принаймні, якщо вам подобається наш інструмент, будь ласка, поставте нам зірку на GitHub і покажіть свою підтримку!

## Подання повідомлень про помилки

Будь ласка, надсилайте будь-які знайдені вами помилки [сюди.](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)
Цей проєкт наразі активно підтримується, і ми будемо раді виправити будь-які повідомлені помилки.

Якщо ви виявите будь-які проблеми (хибні спрацювання, помилки тощо) з правилами Hayabusa, будь ласка, повідомте про них на сторінці issues GitHub hayabusa-rules [тут](https://github.com/Yamato-Security/hayabusa-rules/issues/new).

Якщо ви виявите будь-які проблеми (хибні спрацювання, помилки тощо) з правилами Sigma, будь ласка, повідомте про них на висхідній сторінці issues GitHub SigmaHQ [тут](https://github.com/SigmaHQ/sigma/issues).

## Ліцензія

Hayabusa випускається під ліцензією [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html), а всі правила випускаються під ліцензією [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md).
Ви можете вільно використовувати Hayabusa всередині організації, у SaaS-рішеннях, для консалтингової роботи тощо.
Однак, якщо ви використовуєте Hayabusa у певному типі SaaS-рішення та вносите до нього покращення, ми просимо вас зробити ці покращення з відкритим вихідним кодом і повернути їх проєкту.

Hayabusa використовує дані GeoLite2, створені MaxMind, доступні за адресою [https://www.maxmind.com](https://www.maxmind.com).
