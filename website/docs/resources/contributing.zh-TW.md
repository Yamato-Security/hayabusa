# 貢獻與支援

## 貢獻

我們非常歡迎任何形式的貢獻。
Pull request、規則建立與範例 evtx 日誌是最好的貢獻方式，但功能請求、回報錯誤等也都非常歡迎。

至少，如果您喜歡我們的工具，請在 GitHub 上給我們一顆星，展現您的支持！

## 錯誤回報

請在[此處](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D)回報您發現的任何錯誤。
本專案目前正積極維護中，我們很樂意修正任何回報的錯誤。

如果您發現 Hayabusa 規則有任何問題（誤判、錯誤等），請至 hayabusa-rules GitHub issues 頁面[此處](https://github.com/Yamato-Security/hayabusa-rules/issues/new)回報。

如果您發現 Sigma 規則有任何問題（誤判、錯誤等），請至上游 SigmaHQ GitHub issues 頁面[此處](https://github.com/SigmaHQ/sigma/issues)回報。

## 授權

Hayabusa 以 [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) 授權釋出，所有規則則以 [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md) 授權釋出。
您可以自由地在內部使用 Hayabusa、用於 SaaS 解決方案、顧問工作等。
然而，如果您將 Hayabusa 用於某種 SaaS 解決方案並對其進行改進，我們請求您將這些改進開源並回饋給本專案。

Hayabusa 使用由 MaxMind 建立的 GeoLite2 資料，可從 [https://www.maxmind.com](https://www.maxmind.com) 取得。
