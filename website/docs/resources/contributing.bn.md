# অবদান ও সহায়তা

## অবদান

আমরা যেকোনো ধরনের অবদানকে স্বাগত জানাই।
পুল রিকোয়েস্ট, রুল তৈরি এবং নমুনা evtx লগ সবচেয়ে ভালো, তবে ফিচার অনুরোধ, আমাদের বাগ সম্পর্কে অবহিত করা ইত্যাদিও খুবই স্বাগত।

অন্তত, আপনি যদি আমাদের টুল পছন্দ করেন তাহলে অনুগ্রহ করে GitHub-এ আমাদের একটি স্টার দিন এবং আপনার সমর্থন প্রকাশ করুন!

## বাগ জমা দেওয়া

আপনি যে কোনো বাগ খুঁজে পান তা অনুগ্রহ করে [এখানে](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D) জমা দিন।
এই প্রকল্পটি বর্তমানে সক্রিয়ভাবে রক্ষণাবেক্ষণ করা হচ্ছে এবং আমরা রিপোর্ট করা যেকোনো বাগ ঠিক করতে আনন্দিত।

আপনি যদি Hayabusa রুলে কোনো সমস্যা (ফলস পজিটিভ, বাগ ইত্যাদি) খুঁজে পান, অনুগ্রহ করে সেগুলো hayabusa-rules GitHub issues পৃষ্ঠায় [এখানে](https://github.com/Yamato-Security/hayabusa-rules/issues/new) রিপোর্ট করুন।

আপনি যদি Sigma রুলে কোনো সমস্যা (ফলস পজিটিভ, বাগ ইত্যাদি) খুঁজে পান, অনুগ্রহ করে সেগুলো আপস্ট্রিম SigmaHQ GitHub issues পৃষ্ঠায় [এখানে](https://github.com/SigmaHQ/sigma/issues) রিপোর্ট করুন।

## লাইসেন্স

Hayabusa [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) এর অধীনে প্রকাশিত এবং সমস্ত রুল [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md) এর অধীনে প্রকাশিত।
আপনি অভ্যন্তরীণভাবে, SaaS সমাধানে, পরামর্শ কাজের জন্য ইত্যাদিতে Hayabusa স্বাধীনভাবে ব্যবহার করতে পারেন।
তবে, আপনি যদি কোনো ধরনের SaaS সমাধানে Hayabusa ব্যবহার করেন এবং এতে উন্নতি করেন, আমরা আপনাকে সেই উন্নতিগুলো ওপেন-সোর্স করতে এবং প্রকল্পে ফিরিয়ে দিতে অনুরোধ করছি।

Hayabusa MaxMind দ্বারা তৈরি GeoLite2 ডেটা ব্যবহার করে, যা [https://www.maxmind.com](https://www.maxmind.com) থেকে পাওয়া যায়।
