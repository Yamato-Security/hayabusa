# योगदान और समर्थन

## योगदान

हम किसी भी प्रकार के योगदान का स्वागत करते हैं।
पुल रिक्वेस्ट, नियम निर्माण और नमूना evtx लॉग सबसे अच्छे हैं, लेकिन फ़ीचर अनुरोध, बग के बारे में हमें सूचित करना, आदि का भी बहुत स्वागत है।

कम से कम, यदि आपको हमारा टूल पसंद आता है तो कृपया GitHub पर हमें एक स्टार दें और अपना समर्थन दिखाएँ!

## बग सबमिशन

कृपया आपके द्वारा पाए गए किसी भी बग को [यहाँ](https://github.com/Yamato-Security/hayabusa/issues/new?assignees=&labels=bug&template=bug_report.md&title=%5Bbug%5D) सबमिट करें।
यह परियोजना वर्तमान में सक्रिय रूप से रखरखाव की जाती है और हमें रिपोर्ट किए गए किसी भी बग को ठीक करने में खुशी होती है।

यदि आपको Hayabusa नियमों के साथ कोई समस्या (false positives, बग, आदि) मिलती है, तो कृपया उन्हें hayabusa-rules GitHub issues पेज पर [यहाँ](https://github.com/Yamato-Security/hayabusa-rules/issues/new) रिपोर्ट करें।

यदि आपको Sigma नियमों के साथ कोई समस्या (false positives, बग, आदि) मिलती है, तो कृपया उन्हें अपस्ट्रीम SigmaHQ GitHub issues पेज पर [यहाँ](https://github.com/SigmaHQ/sigma/issues) रिपोर्ट करें।

## लाइसेंस

Hayabusa को [AGPLv3](https://www.gnu.org/licenses/agpl-3.0.en.html) के तहत जारी किया गया है और सभी नियम [Detection Rule License (DRL) 1.1](https://github.com/SigmaHQ/sigma/blob/master/LICENSE.Detection.Rules.md) के तहत जारी किए गए हैं।
आप Hayabusa का उपयोग आंतरिक रूप से, SaaS समाधानों में, परामर्श कार्य के लिए, आदि के लिए स्वतंत्र रूप से कर सकते हैं।
हालाँकि, यदि आप Hayabusa का उपयोग किसी प्रकार के SaaS समाधान में करते हैं और उसमें सुधार करते हैं, तो हम आपसे अनुरोध करते हैं कि आप उन सुधारों को ओपन-सोर्स करें और परियोजना को वापस दें।

Hayabusa MaxMind द्वारा बनाए गए GeoLite2 डेटा का उपयोग करता है, जो [https://www.maxmind.com](https://www.maxmind.com) से उपलब्ध है।
