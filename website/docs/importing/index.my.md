# Timeline ရလဒ်များ ထည့်သွင်းခြင်းနှင့် ခွဲခြမ်းစိတ်ဖြာခြင်း

## Timeline ရလဒ်များ ထည့်သွင်းခြင်းနှင့် ခွဲခြမ်းစိတ်ဖြာခြင်း

Timeline Explorer တွင် CSV timeline များကို မည်သို့ခွဲခြမ်းစိတ်ဖြာရမည်ကို [ဤနေရာတွင်](timeline-explorer.md) လေ့လာနိုင်ပါသည်။

CSV ဖိုင်များကို Elastic Stack ထဲသို့ မည်သို့ထည့်သွင်းရမည်ကို [ဤနေရာတွင်](elastic-stack.md) လေ့လာနိုင်ပါသည်။

CSV ဖိုင်များကို Timesketch ထဲသို့ မည်သို့ထည့်သွင်းရမည်ကို [ဤနေရာတွင်](timesketch.md) လေ့လာနိုင်ပါသည်။

## JSON ဖော်မတ်ရလဒ်များကို JQ ဖြင့် ခွဲခြမ်းစိတ်ဖြာခြင်း

JSON ဖော်မတ်ရလဒ်များကို `jq` ဖြင့် မည်သို့ခွဲခြမ်းစိတ်ဖြာရမည်ကို [ဤနေရာတွင်](jq.md) လေ့လာနိုင်ပါသည်။
