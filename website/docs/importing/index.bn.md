# টাইমলাইন ফলাফল ইম্পোর্ট ও বিশ্লেষণ

## টাইমলাইন ফলাফল ইম্পোর্ট ও বিশ্লেষণ

Timeline Explorer-এ CSV টাইমলাইন কীভাবে বিশ্লেষণ করবেন তা আপনি [এখানে](timeline-explorer.md) শিখতে পারেন।

Elastic Stack-এ CSV ফাইল কীভাবে ইম্পোর্ট করবেন তা আপনি [এখানে](elastic-stack.md) শিখতে পারেন।

Timesketch-এ CSV ফাইল কীভাবে ইম্পোর্ট করবেন তা আপনি [এখানে](timesketch.md) শিখতে পারেন।

## JQ দিয়ে JSON-ফরম্যাটেড ফলাফল বিশ্লেষণ

`jq` দিয়ে JSON-ফরম্যাটেড ফলাফল কীভাবে বিশ্লেষণ করবেন তা আপনি [এখানে](jq.md) শিখতে পারেন।
