# Hayabusa အကြောင်း

Hayabusa သည် ဂျပန်နိုင်ငံရှိ [Yamato Security](https://yamatosecurity.connpass.com/) အဖွဲ့မှ ဖန်တီးထားသော **Windows event log အမြန် forensics timeline ထုတ်လုပ်ပေးသည့်ကိရိယာ** နှင့် **threat hunting ကိရိယာ** ဖြစ်သည်။
Hayabusa သည် ဂျပန်ဘာသာဖြင့် ["peregrine falcon" (သိမ်းငှက်)](https://en.wikipedia.org/wiki/Peregrine_falcon) ဟု အဓိပ္ပာယ်ရှိပြီး၊ သိမ်းငှက်များသည် ကမ္ဘာ့အမြန်ဆုံးတိရစ္ဆာန်ဖြစ်ကာ အမဲလိုက်ရာတွင် တော်ပြီး လေ့ကျင့်ပေးရလွယ်ကူသောကြောင့် ရွေးချယ်ထားခြင်းဖြစ်သည်။
၎င်းကို memory-safe ဖြစ်သော [Rust](https://www.rust-lang.org/) ဖြင့်ရေးသားထားပြီး၊ ဖြစ်နိုင်သမျှ အမြန်ဆုံးဖြစ်စေရန် multi-threading ကို ပံ့ပိုးကာ၊ v2 correlation rules အပါအဝင် Sigma specification ကို အပြည့်အဝ ပံ့ပိုးပေးသည့် တစ်ခုတည်းသော open-source ကိရိယာဖြစ်သည်။
Hayabusa သည် [upstream Sigma](https://github.com/SigmaHQ/sigma) rules များကို parse လုပ်နိုင်သော်လည်း၊ ကျွန်ုပ်တို့အသုံးပြုပြီး [hayabusa-rules repository](https://github.com/Yamato-Security/hayabusa-rules) တွင် host လုပ်ထားသော Sigma rules များကို rule loading ပိုမိုပြောင်းလွယ်ပြင်လွယ်ဖြစ်စေရန်နှင့် false positives များ လျှော့ချရန်အတွက် အချို့သော ပြောင်းလဲမှုများ ပြုလုပ်ထားသည်။
ဤအကြောင်းအသေးစိတ်ကို [sigma-to-hayabusa-converter repository](https://github.com/Yamato-Security/sigma-to-hayabusa-converter) README ဖိုင်တွင် ဖတ်ရှုနိုင်သည်။
Hayabusa ကို live analysis အတွက် single running systems များပေါ်တွင် run နိုင်သည်၊ offline analysis အတွက် single သို့မဟုတ် multiple systems များမှ log များကို စုဆောင်းခြင်းဖြင့်လည်းကောင်း၊ သို့မဟုတ် enterprise-wide threat hunting နှင့် incident response အတွက် [Velociraptor](https://docs.velociraptor.app/) နှင့်အတူ [Hayabusa artifact](https://docs.velociraptor.app/exchange/artifacts/pages/windows.eventlogs.hayabusa/) ကို run ခြင်းဖြင့်လည်းကောင်း အသုံးပြုနိုင်သည်။
output ကို [LibreOffice](https://www.libreoffice.org/), [Timeline Explorer](https://ericzimmerman.github.io/#!index.md) [Elastic Stack](../importing/elastic-stack.md), [Timesketch](https://timesketch.org/) စသည်တို့တွင် လွယ်ကူစွာ ခွဲခြမ်းစိတ်ဖြာနိုင်ရန် single CSV/JSON/JSONL timeline တစ်ခုအဖြစ် ပေါင်းစည်းပေးမည်ဖြစ်သည်။
