# आउटपुट प्रदर्शन और सारांश

## प्रगति पट्टी (Progress Bar)

प्रगति पट्टी केवल एक से अधिक evtx फ़ाइलों के साथ काम करेगी।
यह वास्तविक समय में उन evtx फ़ाइलों की संख्या और प्रतिशत प्रदर्शित करेगी जिनका विश्लेषण पूरा हो चुका है।

## रंगीन आउटपुट (Color Output)

अलर्ट को अलर्ट `level` के आधार पर रंग में आउटपुट किया जाएगा।
आप कॉन्फ़िग फ़ाइल `./config/level_color.txt` में डिफ़ॉल्ट रंगों को `level,(RGB 6-digit ColorHex)` के प्रारूप में बदल सकते हैं।
यदि आप रंगीन आउटपुट को अक्षम करना चाहते हैं, तो आप `-K, --no-color` विकल्प का उपयोग कर सकते हैं।

## परिणाम सारांश (Results Summary)

कुल इवेंट, हिट वाले इवेंट की संख्या, डेटा कमी मेट्रिक्स, कुल और अद्वितीय पहचानें, सबसे अधिक पहचानों वाली तिथियाँ, पहचानों वाले शीर्ष कंप्यूटर और शीर्ष अलर्ट प्रत्येक स्कैन के बाद प्रदर्शित किए जाते हैं।

### पहचान आवृत्ति समयरेखा (Detection Fequency Timeline)

यदि आप `-T, --visualize-timeline` विकल्प जोड़ते हैं, तो इवेंट आवृत्ति समयरेखा सुविधा पहचाने गए इवेंट की एक स्पार्कलाइन आवृत्ति समयरेखा प्रदर्शित करती है।
ध्यान दें: 5 से अधिक इवेंट होने चाहिए। साथ ही, डिफ़ॉल्ट Command Prompt या PowerShell Prompt पर वर्ण सही ढंग से रेंडर नहीं होंगे, इसलिए कृपया Windows Terminal, iTerm2, आदि जैसे टर्मिनल का उपयोग करें...
