# আউটপুট প্রদর্শন ও সারসংক্ষেপ

## প্রোগ্রেস বার

প্রোগ্রেস বার শুধুমাত্র একাধিক evtx ফাইলের ক্ষেত্রে কাজ করবে।
এটি রিয়েল টাইমে সেই evtx ফাইলগুলির সংখ্যা ও শতাংশ প্রদর্শন করবে যেগুলির বিশ্লেষণ এটি সম্পন্ন করেছে।

## রঙিন আউটপুট

অ্যালার্ট `level` এর উপর ভিত্তি করে অ্যালার্টগুলি রঙিনভাবে আউটপুট করা হবে।
আপনি `level,(RGB 6-digit ColorHex)` ফরম্যাটে `./config/level_color.txt` কনফিগ ফাইলে ডিফল্ট রঙগুলি পরিবর্তন করতে পারেন।
আপনি যদি রঙিন আউটপুট নিষ্ক্রিয় করতে চান, তবে আপনি `-K, --no-color` অপশন ব্যবহার করতে পারেন।

## ফলাফল সারসংক্ষেপ

মোট ইভেন্ট, হিট সহ ইভেন্টের সংখ্যা, ডেটা হ্রাসের মেট্রিক্স, মোট ও অনন্য শনাক্তকরণ, সর্বাধিক শনাক্তকরণ সহ তারিখ, শনাক্তকরণ সহ শীর্ষ কম্পিউটার এবং শীর্ষ অ্যালার্টগুলি প্রতিটি স্ক্যানের পরে প্রদর্শিত হয়।

### শনাক্তকরণ ফ্রিকোয়েন্সি টাইমলাইন

আপনি যদি `-T, --visualize-timeline` অপশন যোগ করেন, তবে Event Frequency Timeline ফিচারটি শনাক্ত করা ইভেন্টগুলির একটি স্পার্কলাইন ফ্রিকোয়েন্সি টাইমলাইন প্রদর্শন করে।
দ্রষ্টব্য: ৫টির বেশি ইভেন্ট থাকতে হবে। এছাড়া, ডিফল্ট Command Prompt বা PowerShell Prompt-এ অক্ষরগুলি সঠিকভাবে রেন্ডার হবে না, তাই অনুগ্রহ করে Windows Terminal, iTerm2 ইত্যাদির মতো একটি টার্মিনাল ব্যবহার করুন...
