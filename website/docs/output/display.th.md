# การแสดงผลลัพธ์และการสรุป

## แถบความคืบหน้า (Progress Bar)

แถบความคืบหน้าจะทำงานเฉพาะเมื่อมีไฟล์ evtx หลายไฟล์เท่านั้น
โดยจะแสดงจำนวนและเปอร์เซ็นต์ของไฟล์ evtx ที่วิเคราะห์เสร็จแล้วแบบเรียลไทม์

## การแสดงผลแบบมีสี (Color Output)

การแจ้งเตือนจะถูกแสดงผลเป็นสีตาม `level` ของการแจ้งเตือน
คุณสามารถเปลี่ยนสีเริ่มต้นได้ในไฟล์ตั้งค่าที่ `./config/level_color.txt` ในรูปแบบ `level,(RGB 6-digit ColorHex)`
หากต้องการปิดการแสดงผลแบบมีสี คุณสามารถใช้ตัวเลือก `-K, --no-color` ได้

## การสรุปผลลัพธ์ (Results Summary)

จำนวนเหตุการณ์ทั้งหมด จำนวนเหตุการณ์ที่ตรวจพบ ตัวชี้วัดการลดข้อมูล จำนวนการตรวจจับทั้งหมดและการตรวจจับที่ไม่ซ้ำกัน วันที่ที่มีการตรวจจับมากที่สุด คอมพิวเตอร์ที่มีการตรวจจับมากที่สุด และการแจ้งเตือนยอดนิยม จะถูกแสดงผลหลังจากการสแกนทุกครั้ง

### ไทม์ไลน์ความถี่ของการตรวจจับ (Detection Fequency Timeline)

หากคุณเพิ่มตัวเลือก `-T, --visualize-timeline` ฟีเจอร์ Event Frequency Timeline จะแสดงไทม์ไลน์ความถี่แบบ sparkline ของเหตุการณ์ที่ตรวจพบ
หมายเหตุ: จำเป็นต้องมีเหตุการณ์มากกว่า 5 เหตุการณ์ นอกจากนี้ ตัวอักษรจะไม่แสดงผลอย่างถูกต้องบน Command Prompt หรือ PowerShell Prompt เริ่มต้น ดังนั้นโปรดใช้เทอร์มินัลอย่าง Windows Terminal, iTerm2 ฯลฯ...
